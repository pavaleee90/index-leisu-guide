# index-leisu-guide

## 项目简介

`index-leisu-guide` 是一个用于归档和发布独立 HTML 页面的仓库。本项目不针对任何特定网站或域名，仅作为存放和展示各类静态 HTML 资源的空间。

仓库中的页面可能包含信息展示、工具页面、测试页面或其他独立功能页面，所有内容均以 HTML 文件形式存在，可直接通过浏览器访问。

## 目录结构

```
.
├── README.md          # 本说明文件
├── pages/             # 主要页面存放目录（可选）
├── assets/            # 公共资源目录（CSS/JS/图片等，可选）
└── index.html         # 仓库首页或导航页（可选）
```

目录结构可能随归档内容调整，具体以仓库实际文件为准。

## 页面归档说明

- 每个 HTML 文件均为独立页面，不依赖外部服务或后端。
- 页面可能引用本地资源（如 CSS、JavaScript、图片），资源文件通常存放在 `assets` 或页面同级目录下。
- 归档页面内容可能随时间更新或替换，历史版本可通过 Git 提交记录查看。
- 本仓库不保证所有页面长期可用或兼容所有浏览器。

## 使用方式

你可以通过以下方式浏览仓库中的页面：

1. **直接查看源代码**：在仓库文件列表中点击任意 `.html` 文件，GitHub 会渲染并显示页面内容。
2. **克隆到本地**：使用 `git clone` 将仓库下载到本地，然后用浏览器打开 HTML 文件。
3. **通过 GitHub Pages 访问**（如已启用）：访问对应的 Pages 地址即可在线浏览页面。
